Список сотрудников компании
Данный проект представляет собой простое приложение для управления списком сотрудников компании. Приложение разработано на Python с использованием библиотеки Tkinter для создания графического интерфейса пользователя и SQLite для хранения данных.

Описание функционала
Приложение предоставляет следующий функционал:

Добавление нового сотрудника, включая полное имя, номер телефона, адрес электронной почты и заработную плату.
Редактирование существующей записи сотрудника, включая изменение полного имени, номера телефона, адреса электронной почты и заработной платы.
Удаление записи о сотруднике.
Поиск сотрудников по полному имени.
Отображение данных в виджете Treeview.
Запуск приложения
Для запуска приложения, убедитесь, что у вас установлен Python. Затем выполните следующие шаги:

Склонируйте репозиторий или загрузите файлы проекта.

Установите необходимые библиотеки, если они еще не установлены, используя pip:

shell
Copy code
pip install tkinter
Запустите главный файл main.py:

shell
Copy code
python main.py
Классы и их функциональность
Main
Класс Main представляет главное окно приложения.
Отвечает за отображение списка сотрудников, добавление, редактирование, удаление и поиск записей.
Инициализирует и управляет базой данных SQLite.
Child
Класс Child открывает окно для добавления нового сотрудника.
Позволяет вводить данные сотрудника, включая полное имя, номер телефона, адрес электронной почты и заработную плату.
Сохраняет введенные данные в базе данных.
Update
Класс Update открывает окно для редактирования существующей записи сотрудника.
Позволяет изменять полное имя, номер телефона, адрес электронной почты и заработную плату сотрудника.
Обновляет запись в базе данных.
Search
Класс Search открывает окно для поиска сотрудников по полному имени.
Позволяет пользователю ввести полное имя для поиска.
Отображает результаты поиска на основе введенного имени.
DB
Класс DB управляет базой данных SQLite.
Создает таблицу employees, где хранятся данные о сотрудниках (полное имя, номер телефона, адрес электронной почты и заработная плата).
Вставляет, обновляет, удаляет и извлекает данные из базы данных.
Зависимости
Python 3.x
Библиотека Tkinter (обычно входит в стандартный набор библиотек Python)
